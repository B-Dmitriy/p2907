
### Required dotenv variables

***App***
* URL: string
* PORT: number

***Secrets***
* JWT_ACCESS_SECRET: string
* JWT_REFRESH_SECRET: string

***Database connection***
* DATABASE_HOST: string
* DATABASE_PORT: number
* DATABASE_NAME: string
* DATABASE_USER: string
* DATABASE_PASS: string

***Mail connection***
* MAIL_HOST: string
* MAIL_PORT: number
* MAIL_USER: string
* MAIL_PASS: string
* MAIL_REDIRECT: string
